# Page Topology — vanillasomethin.framer.website

## Page: / (Homepage — index.html)

### Global
- **Background:** #ebe8e5 (warm cream/beige)
- **Font:** Space Grotesk (primary), Aboreto, Fragment Mono, The Seasons Bold, Halimun Regular
- **Smooth scroll:** Lenis (`class="lenis"` on html element)
- **Page layout:** Single column, sections stack vertically
