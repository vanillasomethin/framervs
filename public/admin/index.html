Quick Setup Notes

  1. Keep index.html, admin.js, and cms-loader.js in public/admin/.
  2. Create a GitHub Personal Access Token with repo scope.
  3. Open https://your-domain.com/admin to access the panel.
  4. Change the admin password inside public/admin/admin.js.
